Upload ảnh (drag & drop)

Chú ý: để upload vào repo private, bạn cần Personal Access Token (PAT) có scope repo. Tạo token trên GitHub -> Settings -> Developer settings -> Personal access tokens. Dán token vào ô dưới (chỉ lưu trong session trình duyệt).

Kéo thả file ảnh vào đây hoặc nhấn để chọn. (Hỗ trợ nhiều file)

Uploaded files sẽ được thêm vào thư mục images// trong repo để khỏi trùng tên. Sau khi upload xong, bạn cần cập nhật file gallery.json (mình sẽ gửi hướng dẫn tự động cập nhật nếu muốn).
